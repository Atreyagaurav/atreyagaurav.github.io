* Introduction

The Locus project is for analysis of extreme precipitation patterns in the United States. We're developing the Algorithms to extract, analyse and visualize the precipitation patterns for basins based on their Hydrologic Unit Code (HUC), so that it can be applied to any watershed.

Git repo: https://github.com/Atreyagaurav/locus-code-usace

Data used:
- Livneh Dataset for Precipitation
- USGS dataset for Water boundary
